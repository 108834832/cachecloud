## CacheCloud功能介绍

### 主要功能分以下五个模块:
+ wiki管理：系统介绍、FAQ、系统接入、系统功能、功能手册、常见问题
+ Redis搭建：环境初始化、实例部署安装、类型架构支持
+ 客户端接入：Java-SDK接入、客户端监控、其他语言接入；
+ 运维管理：宿主环境、资源管理、应用审计、应用运维、应用质量监控、应用诊断扥西
+ 弹性伸缩：资源评估、垂直伸缩、水平伸缩、外部接入；
+ 统计监控：指标采集、应用统计、节点统计、机器统计、监控报警、问题诊断；
